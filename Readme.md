## 🛠 How It Works

> Here's a step-by-step flow of your user's experience

1. **Start the Bot**  
   - Click "Start" to initialize wallet, bonus & referral system

2. **Deposit with BharatPe (UPI)**  
   - Scan the QR code and pay via UPI → funds are auto-detected and added to wallet

3. **Use SMM Services**  
   - Choose between views or reactions → enter link → confirm → done!

4. **Refer & Earn**  
   - Share your referral link → get commission on every deposit

5. **Withdraw Earnings**  
   - Enter amount & wallet → approved by admin → funds released

6. **Live Chat Support**  
   - Users can connect directly to admin (only one user at a time)

## ⚙️ Technologies Used

- `Bots.Business` — Scripting platform for Telegram bots  
- `Telegram Bot API` — Messaging, forwarding, UI interaction  
- `BharatPe` — UPI payment support (token-based validation)  
- `GreatSMMPanel` — Optional SMM API support for auto-orders  
- `ResourcesLib`, `Random`, `Cooldown`, `Lang`, `DateTimeFormat` — For logic, localization & cooldown handling
